
# Linux Commands Cheat Sheet

---

## 📁 File System

| Command | Usage |
|---------|-------|
| `cd` | Change the directory |
| `pwd` | Print the current working directory |
| `ls` | List all files and folders |
| `touch` | Create a new empty file |
| `rm` | Remove a file |
| `mv` | Move or rename a file |
| `cat` | Display the contents of a file |
| `nano` | Open a file to write or edit text |
| `grep` | Search for a pattern inside a file or log |
| `wc -l` | Count the number of lines in a file |
| `df` | Show available disk space on the system |
| `chmod +x <file>` | Make a file executable before running it |
| `man` | Open the manual/help page for any command |

---

## ⚙️ Process Management

| Command | Usage |
|---------|-------|
| `ps` | Show a snapshot of current processes |
| `ps aux` | Show all running processes for all users |
| `top` | Real-time monitoring of running processes |
| `htop` | Interactive real-time process viewer |
| `kill` | Terminate a process using its PID |
| `nohup` | Run a process in the background, even after closing the terminal |
| `uname -r` | Print the kernel release and system info |

---

## 🌐 Networking Troubleshooting

| Command | Usage |
|---------|-------|
| `ping <website>` | Test network connectivity to a host |
| `ip addr` | Find the IP address of the system |
| `dig` | Query DNS name servers for domain information |
| `curl` | Transfer data to or from a server using HTTP, HTTPS, FTP, and more |
