# 🚀 90 Days DevOps Journey: Full Stack Developer → SRE

## 📌 Overview
This repository documents my **90-day journey transitioning from a Full Stack Developer to a Site Reliability Engineer (SRE)**.

The goal is to build strong expertise in **DevOps, Linux systems, cloud infrastructure, automation, and reliability engineering**, while leveraging my existing full-stack background.

---

## 🎯 Goals

- Transition from **Full Stack Developer → SRE**
- Gain hands-on experience with **DevOps tools & practices**
- Master **Linux, Networking, CI/CD, and Cloud**
- Build and deploy **production-grade systems**
- Understand **monitoring, scalability, and reliability**
- Create real-world projects showcasing SRE practices

---
